# Raw ACPI Definitions

Contains the given ACPI definitions for its pointers, tables, etc., given at [The UEFI ACPI specs](https://uefi.org/specs/ACPI/6.6/).

This repository's reason to exist is for much more low-level raw control over ACPI configuration, basically leaving it up to the OS respectively.
